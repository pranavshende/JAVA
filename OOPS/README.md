# 🚀 Java OOPs Complete Practice Repository

Welcome to the ultimate **Java Object-Oriented Programming (OOPs)** learning module. This repository is designed to take you from a **Beginner** to an **Advanced/Production-Level** Java developer by mastering the core pillars of OOPs.

---

## 📁 Repository Structure

The repository is organized into logically sequenced modules, each focusing on a specific OOPs concept:

| Folder | Topic | Description |
| :--- | :--- | :--- |
| `v01_Basics` | Introduction to OOPs | Core concepts, procedural vs OOP. |
| `v02_Classes_Objects` | Classes & Objects | Defining blueprints and creating instances. |
| `v03_Constructors` | Constructors | Default, Parameterized, Copy constructors, and Overloading. |
| `v04_Encapsulation` | Encapsulation | Data hiding, Getters/Setters, and shallow vs deep immutability. |
| `v05_Inheritance` | Inheritance | Types of inheritance (Single, Multilevel, Hierarchical). |
| `v06_Polymorphism` | Polymorphism | Method Overloading (Compile-time) & Overriding (Runtime). |
| `v07_Abstraction` | Abstraction | Abstract classes and hiding implementation details. |
| `v08_Interfaces` | Interfaces | Multiple inheritance, Default/Static methods, and Functional Interfaces. |
| `v09_Packages` | Packages | Organizing code, namespace management, and imports. |
| `v10_Access_Modifiers` | Access Modifiers | `private`, `default`, `protected`, and `public`. |
| `v11_Static_Keyword` | Static Keyword | Memory management, static blocks, methods, and variables. |
| `v12_Final_Keyword` | Final Keyword | Final variables, methods, and classes. |
| `v13_This_Super` | This & Super | Referencing current vs parent instances. |
| `v14_Method_Overloading_Overriding` | Overloading vs Overriding | Deep dive into method signatures and dynamic dispatch. |
| `v15_Object_Class_Methods` | Object Class | `toString()`, `equals()`, `hashCode()`, `clone()`, `finalize()`. |
| `v16_Inner_Classes` | Inner Classes | Member, Local, and Static Nested classes. |
| `v17_Anonymous_Classes` | Anonymous Classes | Implementing interfaces/abstract classes on-the-fly. |
| `v18_Enums` | Enumerations | Custom enums with fields, constructors, and methods. |
| `v19_Annotations` | Annotations | Built-in vs Custom annotations. |
| `v20_Exception_Handling` | Exception Handling | `try-catch-finally`, `throw`, `throws`, and Custom Exceptions. |
| `v21_File_Handling_OOP` | OOP File Handling | Using Streams and OOP principles for I/O. |
| `v22_Design_Patterns` | Design Patterns | Singleton, Factory, Builder, and Observer. |
| `v23_SOLID_Principles` | SOLID Principles | S, O, L, I, D principles with practical examples. |
| `v24_Real_World_Examples` | Mini Systems | Bank, Student, and Vehicle management systems. |

## 📌 Features of Each Module
Every concept folder contains:
- **Basic Examples**: Quick start with the syntax.
- **Intermediate Examples**: Adding logic and complexity.
- **Advanced/Real-World Examples**: How it's used in industry.
- **Edge Cases**: Common pitfalls and how to avoid them.
- **Interview Scenarios**: Questions and coded answers focusing on frequently asked topics.
- **Line-by-line Comments**: Detailed explanation of logic and "Why".

## 🚀 How to Run
### Prerequisites
- **JDK 17 or higher** (Recommended for latest features like Records/Sealed classes)
- **IDE**: IntelliJ IDEA, Eclipse, or VS Code.

### Compilation & Execution
1. Navigate to the desired folder:
   ```bash
   cd OOPs/02_Classes_Objects
   ```
2. Compile the Java file:
   ```bash
   javac ClassesAndObjectsDemo.java
   ```
3. Run the compiled class:
   ```bash
   java ClassesAndObjectsDemo
   ```

## 🎯 Learning Roadmap
1. Start with `01_Basics` and `02_Classes_Objects`.
2. Master `03_Constructors` before moving to `04_Encapsulation`.
3. Understand `05_Inheritance` and `06_Polymorphism` as the foundation of reusability.
4. Dive into `07_Abstraction` and `08_Interfaces` for design-level thinking.
5. Explore `10_Access_Modifiers` to understand visibility.
6. Progress to `22_Design_Patterns` and `23_SOLID_Principles` for professional architecture.
