# ☕ Java Master Practice Repository

Welcome to the **Comprehensive Java Mastery Repository**. This repository is a one-stop-shop for learning Java, ranging from **Basic Syntax** to **Advanced JVM Internals** and **System Design Patterns**.

Each module is designed as a standalone practice unit with production-level code, detailed comments, and real-world scenarios.

## 🚀 The Main Runner
The repository includes a central test runner: `Main.java`. This file dynamically executes key demonstrations from **all 26+ modules** to verify the state of the repository.

### How to Run:
1. **Ensure you are in the root directory:**
   ```bash
   cd d:/Java
   ```
2. **Compile the Main class:**
   ```bash
   javac Main.java
   ```
3. **Run the program:**
   ```bash
   java Main
   ```
